# Shopware Tools

This repository contains utilities and tools for working with Shopware, aimed at simplifying and streamlining Shopware development and operations.

## Tools

### 1. **Cache Analyser**
`cacheAnalyser.sh` is a shell script designed to analyze cache usage and performance in your Shopware projects. It helps you identify bottlenecks and optimize your application's caching strategy.
